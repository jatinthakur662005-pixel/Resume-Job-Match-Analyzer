# Resume-Job-Match-Analyzer

An NLP-powered tool that scores resume-job description compatibility using Sentence-BERT embeddings and highlights missing skills — helping job seekers tailor their resumes before applying.

## Overview
This project analyzes how well a resume matches a given job description by computing semantic similarity 
between the two texts using Sentence-BERT embeddings, going beyond simple keyword matching. It also identifies 
matching and missing skills to help users understand their fit for a role.

## Features
- Extracts text from resume PDFs
- Computes match score using Sentence-BERT semantic embeddings
- Identifies matching and missing skills
- Interactive Streamlit web app for live testing

## Tech Stack
Python, Sentence-Transformers (BERT), scikit-learn, PyPDF2, Streamlit
